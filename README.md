# 📌 Gerenciador de Tarefas

Um projeto desenvolvido para facilitar a organização e acompanhamento de tarefas, garantindo uma gestão eficiente e intuitiva.

## 🚀 Funcionalidades

- **Criação e organização de tarefas** 📌  
  - Permite adicionar tarefas e categorizá-las com etiquetas.  
- **Armazenamento Local** 💾  
  - As tarefas ficam salvas entre sessões, evitando perda de informações.  
- **Marcação de conclusão** ✅  
  - Alterna status das tarefas e exibe progresso de conclusão.  
- **Design Responsivo** 📱  
  - Adaptado para diferentes dispositivos, garantindo usabilidade em telas menores.  
- **Dashboard de progresso** 📊  
  - Exibe tarefas concluídas de forma dinâmica.  

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura da aplicação  
- **CSS3**: Estilização e responsividade  
- **JavaScript**: Lógica da aplicação e manipulação de eventos  
- **LocalStorage**: Persistência dos dados entre sessões
